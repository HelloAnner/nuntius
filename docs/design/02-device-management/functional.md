# 设备管理：功能设计

> 实现标记：本文描述目标设计；当前 `0.1.0` 已实现项和后续边界以 [后端实现状态](../../implementation-status.md) 为准。

## 1. 目标与范围

让 Owner 清晰管理所有安装 Nuntius Agent 的工作电脑，并准确知道每台设备当前是否可以执行 Codex 操作。

本模块负责设备登记、命名、状态、能力、版本、撤销和摘要展示；设备身份凭证由身份模块负责，连接实现由设备隧道模块负责。

## 2. 设备信息

每台设备展示：

- 用户自定义名称。
- 操作系统和架构。
- CLI/Agent 版本。
- Codex CLI/App Server 版本。
- 当前状态和最后在线时间。
- App Server 健康状态。
- 项目数量、活跃 Thread 数和待审批数量摘要。
- 历史同步进度、partial Thread 数和最后成功同步时间。
- 当前 WS/WSS 传输安全状态和目录浏览能力。
- 是否需要升级或重新配对。

不默认上报主机名、用户名、IP 明文和硬件序列号。设备名称由用户确认。

## 3. 设备生命周期

```text
pairing -> syncing -> online
                 ├-> degraded
                 └-> offline
offline -> syncing -> online
任意非终态 -> revoked
```

### 3.1 配对中

设备已消费配对码，但尚未完成首次 WS/WSS 连接和状态同步。页面显示“正在接入”，不能远程执行。

### 3.2 同步中

设备连接已建立，正在上传项目索引、核对命令/历史游标和 App Server 状态。同步完成前不接受新的危险命令。

### 3.3 在线

WS/WSS 已认证、心跳正常、控制同步完成，且 Agent 本地存储可写。历史回填可以在 online 后以低优先级继续；App Server 是否可执行通过独立健康字段表示。

### 3.4 降级

Agent 在线，但存在以下一种情况：

- App Server 不可用。
- 本地数据库不可写。
- 事件 outbox 严重积压。
- Codex 版本不兼容。
- 项目根目录大量失效。

页面必须说明受影响功能，不能只显示笼统红点。

### 3.5 离线

没有有效连接。展示 Server 已保存的设备/项目索引和完整会话历史，禁止目录浏览、创建 Turn 和提交审批。

### 3.6 已撤销

设备不能重新连接，从默认列表隐藏；保留审计记录。重新使用需重新配对并获得新 `device_id`。

## 4. 用户功能

- 查看设备列表并按在线、最近使用、名称排序。
- 修改设备显示名称。
- 打开设备详情。
- 查看设备能力和版本兼容提示。
- 手动请求刷新设备摘要。
- 撤销设备。
- 查看设备最后错误和诊断入口。

第一版不提供远程关机、重启电脑或执行任意系统命令。

## 5. 在线判断

- “在线”只表示当前认证连接已经完成同步。
- 仅收到 TCP 连接但未认证不能显示在线。
- 只收到心跳但 App Server 异常时显示降级。
- Server 重启期间所有设备短暂显示重连中，不能立即判为长期离线。
- 同一设备出现两个连接时只允许最新 epoch 成为 active。

## 6. 设备撤销确认

撤销前页面展示：

- 设备名称。
- 最后在线时间。
- 撤销后远程控制立即失效。
- 本地文件和会话不会删除。
- 若要重新接入必须重新配对。

撤销是安全操作，不提供一键恢复。

## 7. 异常场景

- 首次配对后设备未上线：显示配对完成但等待连接，并提供 CLI 诊断命令。
- 设备版本过旧：允许只读状态同步，禁止不兼容写操作。
- 设备名称冲突：允许重名，但列表同时展示 OS 和短设备 ID。
- 设备时钟异常：显示警告，不用设备时间判断在线。
- Agent 连接抖动：UI 使用短暂重连状态，避免在线/离线快速闪烁。

## 8. 验收标准

1. 同一用户可以管理多台在线或离线设备。
2. 在线状态反映认证和同步结果，而不只是 Socket 存在。
3. App Server 异常时设备显示降级及具体原因。
4. 设备撤销后旧连接和旧凭证失效。
5. 离线设备不能接收新的副作用命令。
6. 设备版本不兼容时页面给出明确升级动作。
7. 设备摘要数据标记更新时间和是否 stale。
8. 设备离线不影响 Server 已同步历史的 complete/partial 判断；在线状态与历史完整度分开显示。
9. HTTP/WS 连接显示为 online 时也必须同时标记 insecure。
